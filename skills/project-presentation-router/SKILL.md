# Project Presentation Router

Use after the Future Design method outputs are complete.

Read first:
- Final proposal
- Backcasting roadmap
- Tomorrow headline
- Provotyping card
- Course requirements

Method:
1. Build or update `project-input-package.md`.
2. Identify the project type: platform, public service, product, social innovation, future narrative, strategy, or curriculum.
3. Recommend a minimum deliverable set and an enhanced deliverable set.
4. Select production skills in sequence.
5. For each selected skill, name the input it must read and the output it must save.

Default minimum set:
`documents -> imagegen -> Presentations`

Enhanced options:
`playwright` for interactive prototypes, Figma skills for design files, `xyq-nest-skill` for video, `spreadsheets` for plans, and deploy skills for online publishing.

Output:

```text
## Production Route
Project type:
Minimum set:
Enhanced set:
Skill sequence:
Required inputs:
Expected files:
Archive folders:
```
