# Provotyping Card Builder

Read first: Interpretation, Signal + Challenge Pair, and Local Challenge.

Method:
1. Create a concept that changes a relationship, service, institution, product, or practice.
2. Describe who uses it and how it works.
3. Include risks and ethical tensions.
4. Define what the prototype should provoke or test.

Output:

```text
## Provotyping Card
Name:
One-line concept:
Primary users:
Stakeholders:
How it works:
Future assumption being tested:
What it provokes:
Risks / ethics:
Smallest prototype:
```
