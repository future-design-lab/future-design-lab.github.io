# Future Design Lab Space Camp Docs

This repository contains a documentation-style project homepage for the International Space Art & Design Collaborative Camp.

## Structure

- `index.html`: main documentation homepage
- `styles.css`: responsive documentation-site styling
- `scripts.js`: active sidebar navigation
- `assets/hero`: images extracted from the bilingual camp deck
- `assets/workshop`: rendered workshop-flow pages from the PDF
- `materials/originals`: original PPTX, DOCX, and PDF source files
- `materials/extracted`: extracted text used to structure the site

## Local Preview

From this folder:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
